# Drummer_Scratch

**Link para o Repositório:** [https://scratch.mit.edu/projects/1247253440/](https://scratch.mit.edu/projects/1247253440/)

**Descrição do Projeto:**

O **Drummer_Scratch** é um jogo simples desenvolvido no Scratch onde o jogador precisa acertar uma bateria. A cada acerto, um ponto é adicionado ao contador.
---

## Como Jogar:

1. pressione espaço e up
2. O objetivo do jogo é acertar a bateria para marcar pontos.
3. Cada acerto na bateria incrementa o contador de pontos em +1.
4. Quando o contador de pontos chega a **10**, uma mensagem de fim de jogo é exibida.

---

## Funcionalidades:

- **Contagem de Pontos:** O jogador ganha 1 ponto a cada acerto na bateria.
- **Tela de Fim de Jogo:** Quando o jogador atinge 10 pontos, uma mensagem pop-up é exibida.
- **Reinício Automático:** Após a mensagem de fim de jogo, o jogo reinicia automaticamente para uma nova rodada.

---

## Estrutura do Projeto:

O projeto é composto por três principais **scripts**:

1. **Script da jogador:** 
   - Responsável por detectar quando o jogador acerta a bateria.
   - Controla a movimentação e a interação com os outros objetos no jogo.

2. **Script da Bateria:**
   - Responsável por detectar quando o jogador acerta a bateria e registrar o ponto.
   - Define a posição e a animação da bateria.

3. **Imagem de Fim de Jogo:**
   - Exibe a mensagem de fim de jogo quando o jogador atinge 10 pontos.
   - Controla o momento em que a tela é reiniciada.

---

## Tecnologias Usadas:

- **Scratch:** Plataforma para desenvolvimento do jogo.

---

## Como Rodar o Projeto:

1. Abra o projeto no **Scratch** utilizando o arquivo `.sb3`.
2. Certifique-se de que todas as variáveis e scripts estão corretamente configurados no Scratch.
3. Inicie o jogo clicando na bandeira verde.
